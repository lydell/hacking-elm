# Hacking Elm

To run the slides:

1. `npm ci`
2. `npm start`

Preparations:

1. Open the following files in your editor:

   1. UpperCase.elm
   2. NormalizeNaive.elm
   3. Normalize.elm
   4. Sort.elm
   5. LocaleCompare.elm
